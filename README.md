# RPG Game

This JavaScript website is a "RPG" game. 

The commands are :
- Right arrow : Turn clockwise ⤵️
- Left arrow : Turn anticlockwise ⤴️
- Enter : Validate the move ✅

## Main technology & concept

- JavaScript
- DOM
- `addEventListener`
- `event` parameter
- `for` loop
